## Find the output of the following pseudocode

### Pseudocode:

```
Integer m, n
Set m = 9, n = 6
m = m + 1
n = n - 1
m = m + n
if (m > n)
    print m
else
    print n
end if
```

---

### Python Code:

```python
m, n = 9, 6
m = m + 1 
n = n - 1 
m = m + n 

if m > n:
    print(m)
else:
    print(n)
```

---
 Initial Values: `m = 9`, `n = 6` 

---

### Final Output: **`15`**
